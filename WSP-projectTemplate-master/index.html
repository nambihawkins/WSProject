Header

What I hope to practice and/or learn

I haven't had much practice with adding icons into a design, so I hope to learn a bit more about that. Another issue I have is floating. In this header the contact information is floated to the left and the search bar is floated to the right. I'd to learn how to float multiple items and clear items that aren't meant to be floated.

Relation to degree program

Every "good" web site has some form of navigation. Usually that navigation is found in the header. Therefore, it is imperative to learn how to create a functioning and presentable header. It will be the first thing a visitor looks for when opening a page.

CTA

What I hope to practice and/or learn

There's a strategy behind where to place a CTA that I'm still learning. I know that it should be one of the first things a user notices. First of all, I'd like to practice creating a CTA button and designing it.

Relation to degree program

The call to action is supposed to stand out with out clashing or distracting the user too much. The CTA is the most important part of a site because it allows the user to carry through the task that the site is advertising.
